# SUPERVAN — Exercice technique Mobile

## Setup

```bash
git clone https://github.com/supervan-tech/test_mobile.git
cd test_mobile
npm install
npx expo start
```

Scannez le QR code avec Expo Go ou appuyez sur `w` pour le web.

## Structure

```
src/hooks/useOfflineQueue.ts    ← Votre exercice
src/reference/                  ← Extraits de code de l'app actuelle
src/services/mockApi.ts         ← API simulée (ne pas modifier)
```

## Exercice

Implémentez le hook `useOfflineQueue`.
Les instructions détaillées vous sont fournies séparément.
